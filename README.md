# Flask-URL-Shortner
A basic url shortner built with Flask + Semantic UI

In order to test this site out, you'll need the following installed:

[Python](https://www.python.org/)

[MongoDB](http://mongodb.org/)

#####Python Packages
```
Flask
Flask-WTF
Flask-PyMongo
requests
```

Create a database named links using MongoDB, and insert an empty document using mongo. Navigate to the repository's directory and run python run.py, then go to 127.0.0.1 in your browser.
